CPAP Data Dashboard

Interactive tool for visualizing CPAP therapy data
from SD card exports.

Features
• EDF waveform viewer
• Therapy statistics
• Daily graphs
• Machine identification display

Built using:
HTML
Javascript
Chart.jsFirst github release. alpha
